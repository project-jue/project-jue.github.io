Installation Guide

Jue Language and Runtime repository: https://github.com/project-jue/jue

This guide will walk you through the necessary steps to install and set up the Project Jue environment on your local machine. Follow these instructions carefully to ensure a smooth setup process. We will cover prerequisites, cloning the repository, and setting up the required dependencies.

Prerequisites

Before you begin, ensure you have the following installed on your system:

  • Rust 1.91.1 or higher
  • Git version control system

Installation Steps

First, clone the project repository from GitHub to your local machine. Open your terminal and run the following command:

git clone https://github.com/project-jue/jue.git
cd jue

Next, build the project using Cargo. This will compile the source code and download necessary dependencies.

cargo build --release

Finally, you can run the project:

cargo run --release

Congratulations! You have successfully installed Project Jue. We'll have a quickstart guide eventually, but we're still building the core components. For now, you can explore the codebase and experiment with the core components, as well as the documentation.
